# Portfolio Profesional - Jordy Prada Yanes

Este es mi portafolio profesional personal, diseñado para mostrar mis habilidades como estudiante de Ingeniería de Sistemas, mis proyectos destacados en GitHub y mis certificaciones obtenidas.

## 🚀 Características

- **Diseño Premium:** Interfaz moderna con estética Glassmorphism y degradados profesionales.
- **Sección de Certificados:** Visualización de 11 logros académicos y profesionales con integración de PDFs.
- **Proyectos de GitHub:** Exhibición de repositorios clave con enlaces directos.
- **Totalmente Responsivo:** Adaptado para una navegación fluida en computadoras, tabletas y smartphones.
- **Testimonios:** Carrusel dinámico con referencias personales.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica.
- **CSS3:** Diseño personalizado con variables CSS, Grid y Flexbox.
- **JavaScript:** Funcionalidad para sliders, menús y secciones interactivas.
- **FontAwesome:** Íconos vectoriales de alta calidad.
- **Google Fonts:** Tipografía "Poppins" para una lectura limpia y profesional.

## 📁 Estructura del Proyecto

```text
/
├── index.html        # Página principal
├── css/
│   ├── style.css     # Estilos personalizados
│   └── normalize.css # Reset de estilos
├── Documentos/       # Certificados en formato PDF
├── img/              # Imágenes del proyecto
├── Icons/            # Íconos SVG
└── JS/               # Lógica de interacción
```
