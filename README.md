# Portfolio Profesional - Jordy Prada Yanes

Este es mi portafolio profesional personal, diseñado para mostrar mis habilidades como estudiante de Ingeniería de Sistemas, mis proyectos destacados y mis certificaciones obtenidas. El sitio ha sido optimizado para ofrecer una experiencia de usuario (UX) moderna, rápida y profesional.

## 🚀 Características y Funcionalidades

- **Diseño Premium & Moderno:** Interfaz con estética _Glassmorphism_, degradados suaves y tipografía limpia (Poppins).
- **Modo Oscuro Persistente:** Toggle de tema con persistencia mediante `localStorage` para recordar la preferencia del usuario.
- **Animaciones AOS:** Integración de la librería _Animate On Scroll (AOS)_ para entradas dinámicas y fluidas de cada sección.
- **Optimización SEO:** Implementación de etiquetas _Open Graph (OG)_ y _Twitter Cards_ para previsualizaciones profesionales al compartir en redes (LinkedIn, WhatsApp).
- **Sección de Certificados:** Visualización de 11 logros académicos con acceso directo a sus respectivos PDFs.
- **Proyectos de GitHub:** Exhibición de repositorios clave con enlaces directos y descripciones técnicas.
- **Formulario de Contacto:** Integración funcional con _Formspree_ para recepción de mensajes directos.
- **CV Profesional:** Incluye un documento HTML (`Documentos/CV_Jordy_Prada.html`) optimizado para ser impreso a PDF con un diseño premium.

## 🛠️ Tecnologías Utilizadas

- **HTML5 & CSS3:** Estructura semántica, variables nativas, Flexbox y CSS Grid.
- **JavaScript (Vanilla):** Lógica para el modo oscuro, carrusel de testimonios, acordeones de habilidades y scroll suave.
- **Librerías Externas:**
  - [AOS](https://michalsnik.github.io/aos/) (Scroll Animations)
  - [FontAwesome](https://fontawesome.com/) (Iconografía)
  - [Google Fonts](https://fonts.google.com/) (Tipografía Poppins)

## 📁 Estructura del Proyecto

```text
/

├── css/
```
